Falta un paso para activar la sincronización

Este calendario necesita conectarse a una base de datos Firebase gratuita para guardar los turnos y sincronizarlos entre celulares.

  1. Crea un proyecto gratis en console.firebase.google.com
  2. Activa Firestore Database
  3. Copia tu firebaseConfig y pégalo en el archivo, donde dice PON_AQUI_TU_...

Pide la guía completa paso a paso si la necesitas.
